Etiqueta img VS input type=image

Tutorial

Insertar Imagen: Etiqueta img

Muestra una imagen en la pagina web

- La imagen puede estar guardada localmente en mi proyecto o puede ser un link de otra pagina web

Atributos

src="": Es obligatorio ponerlo. Ruta o link donde esta la imagen

alt="": Es obligatorio ponerlo. Mensaje en caso de no poder mostrar la imagen

input type="image"

Es un boton que contiene una imagen, envia el form y recarga la pagina web. Para que envie el formulario debe esta dentro de la etiqueta form
